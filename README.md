# MagnitudeInML

Этот репозиторий содержит реализацию и эксперименты по применению **метрической величины (magnitude)** в задачах машинного обучения. Работа выполнена в рамках выпускной квалификационной работы на факультете компьютерных наук НИУ ВШЭ (программа "Прикладная математика и информатика").

## О проекте

Магнитуда — это изометрический инвариант метрических пространств, интерпретируемый как «эффективное число точек». Данный проект исследует применение магнитуды в трех ключевых направлениях:

1.  **Функции потерь:** Разработка функций потерь, устойчивых к дисбалансу классов (`CardLoss`) и адаптированных для задач регрессии через динамическое расписание параметра масштаба $t$ (Curriculum Magnitude).
2.  **Оптимизация вычислений:** Ускорение расчета магнитуды с помощью гибридных алгоритмов (декомпозиция на компоненты связности) и итеративных методов.
3.  **Метрики качества:** Анализ `Magnitude Distance` как робастной метрики для сравнения распределений и оценки моделей.

## Структура репозитория (по веткам)

1) CardLoss
Реализация функции потерь на основе магнитуды для классификации. Штрафует модель за разнообразие ошибок в батче, что помогает бороться с дисбалансом классов. Включает скрипты для экспериментов на синтетических данных.

2) CurriculumMagnitude
Динамическое расписание параметра масштаба t (Curriculum Learning). Устраняет нестабильность обучения на ранних этапах и повышает F1-Macro на миноритарных классах по сравнению со статическим подходом.

3) HybridAndDMag
Гибридный солвер для ускорения вычислений через декомпозицию матрицы сходства на компоненты связности. Также содержит реализацию Magnitude Distance и анализ Magnitude Distance как метрики для оценки устойчивости к выбросам.

4) bunch2021weighting
Базовая реализация WeightClassifier (Bunch et al., 2021) для детекции граничных точек и эксперименты на новых данных. Используется как baseline для проверки корректности вычисления весов магнитуды.


5) julia-exps
Эксперименты с итеративными методами (Conjugate Gradient) на языке Julia. Демонстрирует ускорение вычислений до 9x для больших матриц сходства по сравнению с Python-реализацией.
